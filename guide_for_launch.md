# Руководство по запуску

## Запуск анализа:

Для запуска аналитики локально:

```shell or cmd

# Клонируем репозиторий

> git clone https://github.com/nikolaev38/hronos.git

# Переходим в него

> cd htonos

#Устанавливаем список библиотек

> pip install -r requirements.txt



```
